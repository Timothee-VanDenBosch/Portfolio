Wiki Minecraft Skyblock

Ce projet est un site web réalisé en Première lors de la première année en NSI.Ce site web fait en Markdown est publié sur Github.

Ce site hébergé sous Github, est un site dédié aux débutants dans le mode de jeu Skyblock qui lui même fait partie d'un des jeux les plus connus dans le monde, "Minecraft". Celui-ci détient à son effigie 238 millions de vente et cela ne cesse de monter.

Cela vous intéresse ? Vous voulez jeter un coup d'œil ?

Cliquez-ici !

Contenu du Site

Vous êtes toujours là ? C'est une bonne chose, vous savez pourquoi ? Car dans le site mit à disposition au-dessus, je vous apprends mes expériences que j'ai acquises pendant de longues heures de jeu dans le mode Skyblock. Et nous allons dans cette section, évoquez toutes les choses, dont il est possible de trouver dessus et toutes les fonctionnalités du site web, auquel il a à nous apporter.

Pour commencer, vous retrouverez dans ce site web toutes les bases pour commencer une bonne partie de Skyblock, du seau d'eau jusqu'aux techniques de Redstone pour faire de bonnes fermes/machines dans un Skyblock.

Vous apprendrez quelles fermes/machines faire dès le début du jeu. Cela est très important car vous apparaitrez sur une île avec pratiquement aucune ressources, aucune source d'eau à sa disposition. Toutes les choses dont vous disposez au début, sera seulement une petite île dans les airs (généralement une plaine), un coffre avec du contenu et un arbre.

De plus, vous retrouverez dessus une explication sur l'xp ( l'expérience, c'est un acquis exploitable pendant votre session de jeu ), vous aurez aussi une explication rapide sur la pomme de Notch, celle-ci à des facultés avantageuse ( Notch est le créateur de Minecraft ).

Les Entités de Minecraft

Minecraft est constitué d'entités, c'est ce que je vous explique dans une des pages attribué. Cela permet d'avoir des interactions avec l'environnement de Minecraft. Il peut s'agir d'établis, de fours, de bibliothèque. Mais encore des animaux, des monstres et sans oublier les joueurs.

De plus, vous pourrez retrouver un tableau trié avec tous les mobs de Minecraft ( mobs peut signifier soit les animaux et monstres réuni ou tout simplement les monstres du jeu).

Les fonctionnalités du site

Le site possède quelques fonctionnalités telles qu'un système permettant de mettre la page en mode sombre ou en clair. Un bouton permettant de changer la langue de chaque page ( encore en développement ), doté du système responsive pour les téléphones/ tablette, une facilité d'utilisation, avec une lisibité nette. un petit moteur de recherche intégré pour vous aidez dans vos recherches. Et ils vous aies même possible de voir le code des pages via le dépôt situé en haut à gauche.

Conseils

C'est ici que se trouvent les conseils. Pour commencer, je vous recommande une chaîne Youtube de Ilmango, il est anglais donc ses vidéos aussi. Il est spécialisé dans toutes fermes vraiment complexe, je les ai moi même reproduite et ceux sont des fermes/machines vraiment incroyable.

Pour vos problèmes d'xp notamment, il est à votre disposition, il fais toutes sortes de fermes/machines : Bois, Or, Redstone en tout genre, Agriculture, Mobs, Blocs en tout genre tel que la Stone/Cobble. De plus des aides sur certaines fonctionnalités du jeu intéressante, comme ceux qui ressemblent presque à des glitchs.

Je peux aussi vous recommandez Kold Generation, un québécois, pour ceux qui ne sont pas trop à l'aise avec l'anglais. Il fais aussi des fermes/machines très efficace, notamment aussi avec la TNT, glitch pour dupliquer des tapis, Ferme à fer, Machines à Four ( efficace lors de grosse production à vite cuire ) et bien plus à découvrir.
